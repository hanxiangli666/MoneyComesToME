# 生财有术 / MoneyComesToME

这个仓库用于承接我的生财有术学习、商业观察、变现思路和后续小实验。

它不是股票交易系统，也不对应智牛股项目。这里的核心目标是把零散的商业灵感和行动记录沉淀成可复盘、可验证、可继续迭代的项目资产。

## 后续适合放什么

- 生财有术课程或社群方法论的结构化笔记
- AI 工具、内容生产、信息差、自动化变现等方向的实验记录
- 小产品、小脚本、小工作流的验证过程
- 商业案例拆解、复盘、清单和模板
- 能放到个人网站或 GitHub profile 里的公开版项目总结

## 建议目录

```text
notes/          长期笔记和方法论整理
experiments/    小实验、脚本、验证过程
cases/          案例拆解和复盘
templates/      可复用模板、提示词、流程清单
public/         可公开展示的整理版内容
```

## 维护原则

- 先记录真实素材，再整理成文章或项目说明
- 不把网盘提取码、账号、私密链接、付款信息放进仓库
- 公开展示前先做一次脱敏和事实核对
- 项目描述必须和仓库里的实际内容对得上
